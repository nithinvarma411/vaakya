# Vaakya - Intelligent Voice Assistant & AI Agent Platform\n\n🎙️ **Advanced voice transcription and AI-powered task automation platform** with cross-platform capabilities, local processing, and intelligent function calling.\n\n[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://python.org)\n[![FastAPI](https://img.shields.io/badge/FastAPI-Latest-green.svg)](https://fastapi.tiangolo.com)\n[![Ruff](https://img.shields.io/badge/Code%20Quality-Ruff-orange.svg)](https://github.com/astral-sh/ruff)\n[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)\n\n## 🚀 What Vaakya Can Do\n\n### 🎤 **Voice Transcription**\n- **Local Speech-to-Text**: Powered by Faster-Whisper with no internet dependency\n- **Real-time Processing**: Live audio recording and transcription via web interface\n- **Multilingual Support**: Automatic language detection and translation\n- **High Accuracy**: Optimized models with int8 quantization for performance\n\n### 🤖 **Intelligent AI Agent**\n- **Natural Language Understanding**: Interprets user requests and executes appropriate actions\n- **Cross-Platform Operations**: Works seamlessly on macOS, Windows, and Linux\n- **Function Calling**: Automatically chooses the right tools based on user intent\n- **Context Awareness**: Maintains conversation history with 8K token context window\n\n### 🌐 **Web Search & Information Retrieval**\n- **DuckDuckGo Integration**: Search web, news, images, and videos\n- **Smart Results**: Formatted search results with snippets and quick answers\n- **Privacy-First**: No tracking or data collection from search queries\n\n### 📁 **File & Folder Management**\n- **Directory Operations**: List, create, read, write, copy, move files and folders\n- **Cross-Platform Paths**: Intelligent path handling across operating systems\n- **Safe Operations**: Built-in security restrictions for system protection\n- **Batch Processing**: Handle multiple file operations efficiently\n\n### 🚀 **Application Launching**\n- **Fuzzy App Matching**: Launch apps with approximate names (\"notepad\" → finds TextEdit on macOS)\n- **Cross-Platform Discovery**: Automatically discovers installed applications\n- **Smart Suggestions**: Intelligent app matching with 40+ threshold for flexibility\n- **System Integration**: Native OS integration for app launching\n\n### 🔧 **System Operations**\n- **Audio Processing**: Advanced audio handling with ffmpeg integration\n- **Resource Management**: Proper cleanup and memory leak prevention\n- **Configuration Management**: Centralized settings with 9+ specialized configuration classes\n- **Performance Optimization**: Optimized for both CPU and memory usage\n\n## 🏗️ **Architecture
